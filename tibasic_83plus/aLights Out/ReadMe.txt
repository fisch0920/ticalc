=============
INTRODUCTION
=============

The classic game of Lights Out, packed full of features including high scores, customizable 
board sizes, and fast/responsive graphical gameplay.  If you're unfamiliar with Lights Out, 
basically, you have to invert squares on a grid to eventually turn all of the "lights" off.
-- By Fisch2


==============
REQUIREMENTS
==============

1) Send the ALIGHTS.83p file to your calculator
2) Run the ALIGHTS program (listed under the PRGM key)

Compatibility Notes:

* Supported calculators include:  TI-83, TI-83+, TI-83+ Silver Edition, and the TI-84 Plus/SE
* This program is written in TI-Basic and is compatible with MirageOS


=============
INSTRUCTIONS
=============

Use the arrow keys to move your cursor around the grid.  Press 2nd to invert the current square, 
along with the 4 squares directly connected to it.  The goal of the game is to turn all of the lights 
to white.
Pressing CLEAR at any time during gameplay will exit the game.
